An IP Address Tracker web application that allows users to view their current IP address information and search for any IP address or domain to see its geographical location on an interactive map.


## üìå Overview

This project uses the **IP Geolocation API by IPify** to fetch IP address details and **LeafletJS** to display the location on a map. On page load, the app automatically detects and displays the user‚Äôs IP address and location.

Users can also search for any valid IP address or domain name to retrieve key information such as location, timezone, and ISP.

## Features

- Automatically displays the user‚Äôs IP address on page load
- Search for any IP address or domain name
- Displays:
  - IP Address
  - Location (City, Country, Postal Code)
  - Timezone
  - Internet Service Provider (ISP)
- Interactive map showing the IP location
- Responsive design for mobile and desktop
